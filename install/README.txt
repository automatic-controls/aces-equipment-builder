
ACES Equipment Builder

Description
  Desktop application that interfaces with WebCTRL to generate EIKON scripts using custom .logicsymbol libraries.
  Capable of maintaining a shared set of favorite .logicsymbol and .logic-script files.
  Developed for Windows and has not been tested on Mac or Linux operating systems.
